# Release report 10.0

## Result

10.0 is an additive interaction release based on 9.0. It introduces coordinate-correct magnetic drag, visible snap guidance, drop slots, conflict-aware preference, configurable long-press activation, accessibility announcements, a delight preset and a new example.

## Compatibility

Previous versions and public APIs remain exported. No push, tag or pub.dev publication was performed.

## Blocking validation

A public release remains blocked until `flutter analyze`, `flutter test`, web/Android release builds and `flutter pub publish --dry-run` pass on a machine with Flutter installed.
