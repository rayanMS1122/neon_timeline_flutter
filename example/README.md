# neon_timeline_flutter example

This app demonstrates:

- `NeonScheduleTimeline<T>` with application-owned task models.
- Long-press drag-to-reschedule with five-minute snapping.
- Overlap detection and free-time gaps.
- `NeonSlidableTimeline` actions.
- `NeonTimelineDayPager` day switching.
- The `omniverse` visual preset.

Run from this directory:

```bash
flutter pub get
flutter run
```
