# Accessibility 9.0

The public Structured components provide semantic labels for entries, controls,
markers, conflict indicators, delete targets, states, and navigation actions.

The production card adapts its information density to both available height
and text scaling. Secondary information is removed before it can overflow;
the title and time remain visible. Completion, lock, recurrence, and
continuation indicators have dedicated semantic descriptions.

The package continues to support keyboard interaction, high-contrast style
presets, reduced-motion configuration, RTL-compatible layout primitives, and
host-provided localized labels through `StructuredTimelineStrings`.
