# API Changes 7.0

## Added

- `StructuredTimelineStyle`
- `StructuredTimelineStrings`
- `StructuredTimelineInitialScroll`
- `StructuredTimelineEntryDetails<T>`
- `StructuredTimelineMoveDetails<T>`
- `StructuredTimelineView<T>`
- `StructuredTimelinePlanner<T>`
- card, marker, trailing, title, subtitle, progress, time, and duration builders
- async entry, completion, move, delete, and gap callbacks
- `timeline_v7.dart`

## Preserved

- every 6.x planner engine API;
- every 5.x query, recurrence, scenario, and view API;
- every neutral 4.x API;
- the legacy Neon renderer.
