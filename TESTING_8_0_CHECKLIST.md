# 8.0 manual interaction checklist

- Open every version card in the example catalog.
- Long-press and move a normal task with five-minute snapping.
- Drag near both viewport edges and verify controlled auto-scroll.
- Resize start and end handles.
- Verify locked external entries cannot move or resize.
- Trigger a conflict-blocked resize.
- Trigger a simulated save failure and verify the app-owned model remains unchanged.
- Select a free gap and create a task.
- Exercise jump-to-now, jump-to-entry, zoom and keyboard nudge.
- Test light/dark, text scaling, reduced motion, keyboard, mouse and touch.
