# Accessibility

The compact timeline provides:

- focusable entry rows
- visible focus outlines
- semantic title and time-range labels
- screen-reader increment/decrement movement
- custom end-time resize actions
- keyboard move, accelerated move, resize, and direct time editing
- tooltips and semantics for header actions
- status information through text/icons in addition to color
- large invisible drag/resize hit regions
- reduced dependence on precise pointer targeting through snapping and hysteresis

The proportional editor retains keyboard movement, semantic slider values, direct callbacks, 48 × 48 interaction targets, high-contrast themes, RTL support, and reduced-motion-aware behavior.

Validate final semantics and text scaling on target devices because fonts, locale, browser zoom, and platform accessibility bridges affect layout.
