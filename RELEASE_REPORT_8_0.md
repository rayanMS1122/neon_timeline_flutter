# Release report 8.0.0

## Scope

Version 8.0 preserves every previous package generation and introduces a production-oriented orchestration layer for the Structured timeline.

## Delivered

- searchable example catalog for 1.x–8.0 and feature laboratories;
- advanced Structured timeline and planner widgets;
- host-controlled navigation, selection, zoom and nudge controller;
- start/end resizing with snap, bounds and conflicts;
- per-entry asynchronous mutation coordination;
- viewport interval index;
- free-slot suggestion engine;
- compact, comfortable, dense, absolute-time and custom layouts;
- additive builder hooks in the 7.x renderer;
- right-side time-column option;
- benchmark and regression tests;
- migration, architecture, API and performance documentation.

## Release status

No push, tag or publication was performed. Static package validation is documented separately. Flutter analyzer, tests and builds remain mandatory before publication because no Flutter SDK was available in the generation environment.
