# Release Report 9.0

## Result

Version 9.0 is an additive production-composition release over the corrected
8.0.0 source. It preserves legacy Neon, 4.x–8.x APIs, and every previous
example route.

## Main changes

- Public component system for the Structured timeline
- Overflow-resistant adaptive entry card
- Midnight and multi-day segment model
- Proportional, compressed, hybrid, fixed-maximum, and custom gap policies
- Correct gap extents for scrolling and visible-range calculations
- Public scaffold, header, rail, marker, state, action, drag, resize, conflict,
  and entry components
- Whole-card and section-level builders
- Production all-in-one widget and granular composition API
- Component Explorer and retained version gallery
- Warm, Neon, and high-contrast Structured presets
- Additional public detail models

## Release status

Static package validation passed. Flutter analyzer, Flutter tests, Android/Web
release builds, and publish dry-run were not executable in this environment.
No push, tag, or pub.dev publication was performed.
